# Universidad Distrital - Sitio Web Oficial

Página web institucional de la **Universidad Distrital Francisco José de Caldas**, diseñada con tecnologías web modernas para proporcionar información completa sobre programas académicos, servicios y contacto.

## 📋 Descripción del Proyecto

Este proyecto consiste en una página web responsiva y profesional que representa la Universidad Distrital, ofreciendo a estudiantes, padres de familia y el público general una experiencia de navegación intuitiva e informativa.

## ✨ Características Principales

- 🎓 **Información Institucional**: Misión, visión y valores de la universidad
- 📚 **Programas Académicos**: Catálogo de 6 programas de pregrado
- 📊 **Estadísticas Institucionales**: Datos clave sobre estudiantes, docentes y trayectoria
- 📧 **Formulario de Contacto**: Comunicación directa con la institución
- 📱 **Diseño Responsivo**: Optimizado para desktop, tablet y dispositivos móviles
- 🎨 **Interfaz Moderna**: Diseño limpio y profesional con colores institucionales
- ♿ **Accesibilidad**: Navegación clara y estructura semántica

## 🏗️ Estructura de la Página

### Secciones Incluidas

1. **Encabezado (Header)**
   - Logo y nombre de la institución
   - Navegación principal con enlaces a secciones

2. **Sección Hero (Principal)**
   - Bienvenida destacada
   - Llamada a la acción

3. **Acerca de Nosotros**
   - Misión y visión
   - Razones para elegir la universidad

4. **Programas Académicos**
   - Ingeniería de Sistemas
   - Administración de Empresas
   - Educación
   - Ingeniería Civil
   - Derecho
   - Contabilidad

5. **Estadísticas**
   - 15,000+ estudiantes activos
   - 50+ programas académicos
   - 500+ docentes expertos
   - 30+ años de excelencia

6. **Contacto**
   - Información de ubicación y teléfono
   - Correo electrónico institucional
   - Formulario de consultas
   - Enlaces a redes sociales

7. **Pie de Página (Footer)**
   - Información de derechos reservados
   - Lema institucional

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos responsivos con Grid y Flexbox
- **JavaScript**: Interactividad (formularios)
- **Responsive Design**: Media queries para adaptabilidad

## 📱 Características Responsivas

- Diseño fluido que se adapta a pantallas de cualquier tamaño
- Menú de navegación optimizado para dispositivos móviles
- Grid layouts que se ajustan automáticamente
- Tipografía legible en todos los dispositivos

## 🚀 Cómo Usar

1. **Clonar el repositorio**
   ```bash
   git clone https://github.com/MET-UD/PruebaMET.git
   cd PruebaMET
   ```

2. **Abrir en el navegador**
   ```bash
   # Simplemente abre el archivo index.html en tu navegador
   open index.html  # macOS
   # o
   xdg-open index.html  # Linux
   # o
   start index.html  # Windows
   ```

3. **Con un servidor local (opcional)**
   ```bash
   # Usando Python 3
   python -m http.server 8000
   
   # Usando Node.js con http-server
   npm install -g http-server
   http-server
   ```
